# Dynamic Augmentative and Alternative (AAC)
### Authors: Matthew Yurkunas & Kaitlin Moore

# Requirements
Python 3.13.5

# Setup
1. Install dependencies:
    pip install -r requirements.txt
2. Run the app:
    streamlit run app.py

Note: The first time you run the app, it may take a while because the pretrained models need to download.
